# Obsidian CSS Variables 문서 URL (Jina Reader)

이 문서는 Obsidian CSS Variables 공식 문서를 WebFetch 도구로 가져올 때 사용할 Jina Reader URL 목록입니다.

## Jina Reader API 사용법

Jina Reader API는 웹 페이지를 마크다운으로 변환해줍니다:

```
https://r.jina.ai/{원본_URL}
```

WebFetch 도구와 함께 사용하면 최신 문서를 마크다운 형식으로 가져올 수 있습니다.
